# 🐍 贪吃蛇游戏

一个使用HTML5 Canvas和现代CSS技术开发的经典贪吃蛇游戏，具有流畅的动画效果、音效反馈和响应式设计。

## 🎮 游戏特性

### 核心功能
- **经典玩法**：20x20网格游戏区域，蛇吃到食物后长度增加
- **流畅控制**：通过方向键(↑↓←→)控制蛇的移动，响应迅速
- **智能碰撞检测**：撞墙或撞到自身时游戏结束
- **动态难度**：随着得分增加，蛇的移动速度逐渐加快

### 视觉效果
- **现代UI设计**：渐变背景、圆角边框、阴影效果
- **丰富的色彩**：蛇头深灰色、蛇身绿色、食物红色
- **网格显示**：清晰的游戏网格边界
- **动画效果**：按钮悬停、分数增加动画

### 音效系统
- **吃食物音效**：清脆的提示音
- **移动音效**：轻微的移动声音
- **游戏结束音效**：低沉的失败提示

### 游戏状态管理
- **开始界面**：游戏说明和开始按钮
- **游戏进行**：实时显示得分和最高分
- **暂停功能**：空格键暂停/继续游戏
- **结束界面**：显示最终得分和重新开始选项

### 响应式设计
- **自适应布局**：适配不同屏幕尺寸
- **移动端优化**：在小屏幕上自动调整大小
- **触摸友好**：按钮和界面元素适合触摸操作

## 🚀 快速开始

### 文件结构
```
snake-game/
├── index.html      # 主页面文件
├── style.css       # 样式文件
├── game.js         # 游戏逻辑
└── README.md       # 说明文档
```

### 运行方法
1. 下载所有文件到同一目录
2. 用浏览器打开 `index.html` 文件
3. 点击"开始游戏"按钮即可开始

### 控制方式
- **方向键**：↑↓←→ 控制蛇的移动方向
- **WASD键**：WASD 控制方向
- **空格键**：暂停/继续游戏
- **鼠标**：点击按钮开始/重新开始游戏

## 🎯 游戏规则

### 得分系统
- 每吃一个食物获得 **10分**
- 游戏速度会随着得分增加而加快
- 最高分会被保存到本地存储中

### 游戏机制
- 蛇初始长度为1，位于游戏区域中心
- 食物随机生成在空白位置
- 蛇不能反向移动（例如正在向右移动时不能直接向左）
- 撞墙或撞到自身会导致游戏结束

## 🎨 自定义选项

### 游戏配置
在 `game.js` 文件中可以调整以下参数：
```javascript
this.gridSize = 20;      // 网格大小
this.tileCount = 20;     // 网格数量（20x20）
this.gameSpeed = 150;    // 初始游戏速度（毫秒）
```

### 视觉样式
在 `style.css` 文件中可以修改：
- 颜色主题和渐变
- 动画效果和过渡
- 响应式断点
- 字体和排版

### 音效参数
在 `game.js` 的 `createSound` 方法中可以调整：
- 音效频率和持续时间
- 音量大小
- 音效类型
